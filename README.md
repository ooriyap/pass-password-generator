## Overview
**PASS - Secure Password Generator** is a modern, high-performance password generator built with **Astro**, **Alpine.js**, and **Tailwind CSS**. It is designed to be fast, secure, and fully offline-capable (PWA).

The application generates strong, customizable passwords entirely in your browser. **No data is ever sent to a server**, ensuring complete privacy and security.
